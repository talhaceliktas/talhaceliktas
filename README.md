# 💫 About Me:
Hi! I'm a Full Stack Software Engineer who builds scalable and high-performance web applications by bridging innovative front-end designs with robust back-end architectures. 

With solid experience in Next.js, React, and Tailwind CSS, I create dynamic, user-centric interfaces enriched by smooth GSAP animations.
